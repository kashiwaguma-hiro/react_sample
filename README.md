# react_sample

# Run sample
1. Up node
```bash
$ docker-compose up -d
```
2. cd
``` bash
$ docker exec -it reactsample_node_1 bash
$ cd todo-app
$ npm start
```
3. View any browser
```bash
http://localhost:3000
```
4. deploy to gh-pages
```bash
$ pwd
/usr/src/app/todo-app

$ npm run deploy
```
5. show in browser.
```bash
https://kashiwaguma-hiro.github.io/react_sample/
```
